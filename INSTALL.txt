INSTALLATION
------------


Dependencies
============
- Python 3
- Python Imaging Library - PIL (https://github.com/sloonz/pil-py3k)
#- Python Requests >= 0.13.7 (https://github.com/kennethreitz/requests)


Installation
============
For Archlinux there is a package called titulky_com_downloader in AUR:
https://aur.archlinux.org/packages.php?ID=61099

For other linux distributions just run file 'run.sh'
